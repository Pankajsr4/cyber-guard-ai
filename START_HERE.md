## 🐛 IF SOMETHING GOES WRONG

### PowerShell Error?
Run PowerShell as Administrator and execute:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Port Already in Use?
```bash
npx kill-port 3000
```

### Backend Not Running?
```bash
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```
